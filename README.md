# Sales Dashboard (Power BI)

## Overview
This project features an interactive **Sales Analytics Dashboard** built using **Power BI** and the **Superstore Sales dataset**.  
The dashboard provides insights into sales performance across time, regions, and product categories to support data-driven decision making.

## Key Features
- **Total Sales KPI** with dynamic filtering
- **Sales breakdown by Region, Category, and Season**
- **Time Series Analysis** to identify sales trends and seasonality
- **Interactive slicers** for user-driven exploration
- Clean, executive-style visual layout

## Tools & Technologies
- **Power BI**
- **DAX** (for calculated measures)
- **Data Modeling**
- **Data Visualization & Analytics**

## Dashboard Insights
- Identifies top-performing regions and product categories
- Highlights seasonal sales patterns
- Enables quick performance comparison using slicers

## How to Use
1. Download the `Sales_Dashboard.pbix` file.
2. Open it using **Power BI Desktop**.
3. Interact with slicers to explore sales trends and insights.

## Dataset
- Source: Superstore Sales dataset  
- Data includes orders, regions, categories, and sales metrics
